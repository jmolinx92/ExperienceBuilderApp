NMFS ESA Critical Habitat Geodatabase attribute table definitions and values. For more details, please refer to the geodatabase metadata, which is available at the "Download geodatabase" link in the top of this tool.

ID
9 digit unique identifier for each feature in the NMFS ESA Critical Habitat Geodatabase.

Scientific Name
Binomial or trinomial scientific name.
Value formatted as written in the Code of Federal Regulations.

Common Name

Legal common name of species.

Value formatted as written in the Code of Federal Regulations.


Listed Entity

Full text of the ESA listed entity: Species, Subspecies, Distinct Population Segment (DPS), or Evolutionarily Significant Unit (ESU).

Value formatted as written in the Code of Federal Regulations.


Listing Status

Current Endangered Species Act listing status.

Values: Endangered, Threatened


Critical Habitat Status

Status of the critical habitat designation.

Values: Proposed, Final


Unit

Critical habitat unit name.

Value is null if unit names were not specified.


Taxon

Taxonomic unit.

Values: baleen whale, toothed whale, fish, pinniped, marine reptile, invertebrate, plant

Lead Office

Office responsible for the critical habitat Federal Register Notice.

Values: Office of Protected Resources, Alaska Region, West Coast Region, Greater Atlantic Region, Southeast Region, Pacific Islands Region


Federal Register Rule

Federal Register Rule citation for the critical habitat final rule or proposed rule. The citation is linked to the NMFS webpage for the rule.

Volume #, FR, Page Number (Example: 79 FR 39855)


Publication Date

Federal Register Notice publication date.

MM/DD/YYYY


Effective Date

Effective date for a final critical habitat designation.

Value is null for proposed critical habitat.

MM/DD/YYYY format


Length Km

Geodesic length calculated in kilometers.

Lines only.


Area SqKm

Geodesic area calculated in square kilometers.

Polygons only.


Create Date

Date spatial feature was created or geometry was last edited.

MM/DD/YYYY


Notes

Additional information about the feature that is not contained in other fields.


InPort URL

Metadata URL for each layer in InPort, the National Marine Fisheries Service's official metadata catalog.

The metadata record for the entire NMFS ESA Critical Habitat Geodatabase is

https://www.fisheries.noaa.gov/inport/item/65207


HabType

Type of habitat.

Values not standardized.


eCFR

Link to electronic Code of Federal Regulations for designated critical habitat only
